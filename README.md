The Matlab code for our paper :"Soil moisture thresholds for the temperature sensitivity of ecosystem respiration" The fold named dataanalysis.m contains the codes, which are used to identify soil moisture thresholds for the temperature sensitivity of ecosystem respiration from 212 FLUXNET2015 tower EC sites. The fold named RandForest.m is used to analyze the direct relationship between soil moisture thresholds and environmental variables using the random forest method.
